# all-terrain-camper-frank-design-spacer
Frank Design sells a MOLLE pannel for 4 wheel campers on the lift panel. Some ATCs need this spacer to mount cleanly. 

[This](https://frankdesign.pro/products/four-wheel-camper-lift-board-molle-panel) is the panel, designed for four wheel campers. If your lift panel is not 22 1/8" wide you will likely need a 50mm spacer on one edge to mount it on all four sides. It's designed for a #10 standard machine screw to run through it. I aligned my drill with the spacer (printed in nylon) and drilled through the frame rail with a 3/16" bit.
